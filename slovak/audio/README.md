# Audio files

This site is designed for **your own recordings**.

## Where to put files

Put your audio files in these folders:

- `audio/lesson1/` (alphabet letters)
- `audio/lesson2/` (greetings)

The lesson JSON files reference these exact paths, so keep the filenames the same (or edit the JSON).

## Recommended recording/export settings

- Record in a quiet room.
- Keep mic distance consistent.
- Export as **MP3** (recommended) or **WAV**.
- 44.1 kHz or 48 kHz, mono is fine.
- Keep each clip short (0.3–1.5s for letters; 0.5–2.5s for phrases).
